# AI Operating System

## Roles
- ChatGPT = strategy, system design, debugging triage, business planning
- Claude = implementation, file rewrites, code generation
- VS Code = source of truth
- Git = backup and rollback

## Rules
1. Never paste generated modules inside function bodies
2. Always backup core files before edits
3. Commit before major rewrites
4. One terminal only: VS Code terminal
5. Confirm working directory with pwd before commands
